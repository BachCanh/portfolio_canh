## üöÄ Getting Started

### Prerequisites

- Node.js (version 18+)
- npm or yarn

### Installation

1. Clone the repository

   ```bash
   git clone <repository-url>
   cd portfolio_canh
   ```

2. Install dependencies

   ```bash
   npm install
   # or
   yarn
   ```

3. Start the development server

   ```bash
   npm run dev
   # or
   yarn dev
   ```

4. Open your browser and visit `http://localhost:5173`
